# Installation

```bash
npm install @bcc-code/design-tokens
```

## Font

Add the Archivo font to your HTML `<head>`:

```html
<link href="https://fonts.googleapis.com/css2?family=Archivo:wght@200..900&display=swap" rel="stylesheet" />
```
