# BCC Tokens 2.0

Design tokens for building consistent BCC applications.

## Quick Start

```bash
npm install @bcc-code/design-tokens
```

```css
@import "@bcc-code/design-tokens/tailwind";
```

```html
<div class="bg-elevation-surface-default text-default p-200 radius-md">
  Your content
</div>
```
