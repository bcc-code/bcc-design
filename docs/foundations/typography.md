## Font Family

| Token | Tailwind Class | Value |
|-------|---------------|-------|
| `font.archivo` | `font-archivo` | Archivo |

::: info Font Import Required
Import the Archivo font in your project before using `font-archivo`:

```html
<link href="https://fonts.googleapis.com/css2?family=Archivo:wght@400;500;700&display=swap" rel="stylesheet">
```
:::
