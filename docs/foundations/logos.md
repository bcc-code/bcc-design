# Logos

Base URL: `https://design.bcc.no/logos/`

## URL Pattern

**SVG (recommended)**
```
{name}_logo.svg
```

**PNG**
```
{name}_logo_{color}_{size}.png
```

| Color      | Sizes (px)        |
| ---------- | ----------------- |
| dark-green | 32 / 48 / 64 / 72 |
| white      | 32 / 48 / 64 / 72 |

**Example:**
```
https://design.bcc.no/logos/bcc-bergen_logo.svg
https://design.bcc.no/logos/bcc-bergen_logo_dark-green_72.png
```
